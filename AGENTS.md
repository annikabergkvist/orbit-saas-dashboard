## Icons and shadcn-adjacent UI

When adding icons, blocks, or patterns that exist on **[shadcn.io](https://www.shadcn.io)** (for example [Icons → Clarity → Slider Line](https://www.shadcn.io/icon/clarity-slider-line)), **check shadcn.io first** before pulling from other catalogs. Many entries align with **Iconify** collections (same names, e.g. `clarity:slider-line`); prefer matching those glyphs so installs stay consistent with shadcn.io.
